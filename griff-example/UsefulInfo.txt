Create background pixil art in Aseprite:
    -New file:160px144px,Indexed,Transparent
    -Edit Preferences: Grid 8Height 8Width, Background 8x8
    -Export As... png file 

GENERATE BACKGROUND FILES:
cd gdbk/bin/
./png2asset /path/to/background.png -map -use_map_attributes

GENERATE SPRITE FILES:
cd gdbk/bin/
./png2asset ../../Sprites/WalkingGorbDown.png -sw 16 -keep_palette_order
(-sw 16 is defining that the metasprite width is 16 pixels)
See manual here:
https://gbdk.org/docs/api/docs_toolchain_settings.html#png2asset-settings

In the directory containing your Makefile, run:
$make clean
$make
